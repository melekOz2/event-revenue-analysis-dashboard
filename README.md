# 📊 Event Revenue Analysis Dashboard

This project is an event revenue analysis dashboard created using Google Sheets.

## 🚀 Project Purpose
The goal of this project is to analyze event revenues, discount impact and profitability to support data-driven decision making.

## 📌 Key Insights
- Identified the most profitable event
- Calculated revenue loss caused by discounts
- Compared discounted vs non-discounted revenue
- Suggested marketing strategy for next month

## 🛠 Tools Used
- Google Sheets
- Pivot Tables
- Data Analysis
- Dashboard Design
- Data Visualization

## 📈 Dashboard Features
- Revenue comparison chart
- Discount impact analysis
- Most profitable event detection
- Analyst insights section
